# 🚀 Getting Started

## Prerequisites

Before using the workflows, ensure you have:

- n8n Cloud or Self-Hosted Instance
- Google Gemini API Key
- Apify API Token
- Google Account (for Sheets/Gmail)
- Postman (optional, for API testing)

---

# 📥 Installation

Clone the repository.

```bash
git clone https://github.com/Ay7753/n8n-automation-workflow.git
```

Navigate to the project.

```bash
cd n8n-automation-workflow
```

Open your n8n instance and import any workflow JSON file.

---

# ⚙️ Import Workflow into n8n

1. Open n8n.
2. Click **Import from File**.
3. Select the desired workflow JSON file.
4. Configure the required credentials.
5. Save the workflow.
6. Execute and test.

---

# 🔑 Required Credentials

Depending on the workflow, configure:

- Google Gemini API Key
- Apify API Token
- Google Sheets OAuth

- Webhook URLs
